# A

### AJAX
**EN**: Asynchronous JavaScript and XML - technique for updating web pages without reloading.
**NL**: Techniek voor het bijwerken van webpagina's zonder herladen.

### Algorithm
**EN**: Step-by-step instructions to solve a problem.
**NL**: Stap-voor-stap instructies om een probleem op te lossen.

### AUTO_INCREMENT
**EN**: Database feature that automatically increases ID numbers (1, 2, 3...).
**NL**: Database functie die automatisch ID nummers verhoogt.

---

# B

### bcrypt
**EN**: Secure password hashing algorithm. Converts passwords to unreadable text.
**NL**: Veilig wachtwoord hash algoritme. Zet wachtwoorden om naar onleesbare tekst.

### Bootstrap
**EN**: CSS framework for responsive, mobile-first web design.
**NL**: CSS framework voor responsive, mobile-first webdesign.

### Bug Fix
**EN**: Correction of an error in code.
**NL**: Correctie van een fout in code.

---

# C

### CASCADE
**EN**: Database rule: when parent record is deleted, delete related child records.
**NL**: Database regel: wanneer ouder record verwijderd wordt, verwijder gerelateerde kind records.

### Client-side
**EN**: Code that runs in the user's browser (JavaScript).
**NL**: Code die draait in de browser van de gebruiker (JavaScript).

### CRUD
**EN**: Create, Read, Update, Delete - the 4 basic database operations.
**NL**: Aanmaken, Lezen, Bijwerken, Verwijderen - de 4 basis database operaties.

### CSS
**EN**: Cascading Style Sheets - language for styling web pages.
**NL**: Taal voor het stylen van webpagina's.

---

# D

### Database
**EN**: Organized collection of data stored on a server.
**NL**: Georganiseerde verzameling van data opgeslagen op een server.

### DOM
**EN**: Document Object Model - browser's representation of a web page.
**NL**: Document Object Model - browser's representatie van een webpagina.

---

# E

### ERD
**EN**: Entity Relationship Diagram - visual representation of database tables.
**NL**: Entity Relationship Diagram - visuele representatie van database tabellen.

---

# F

### Foreign Key (FK)
**EN**: Column that links to another table's primary key.
**NL**: Kolom die linkt naar de primaire sleutel van een andere tabel.

### Function
**EN**: Reusable block of code that performs a specific task.
**NL**: Herbruikbaar blok code dat een specifieke taak uitvoert.

---

# H

### Hash
**EN**: One-way encryption. Cannot be reversed.
**NL**: Eenrichtings versleuteling. Kan niet worden teruggedraaid.

### HTML
**EN**: HyperText Markup Language - structure of web pages.
**NL**: Structuur van webpagina's.

### HTTP/HTTPS
**EN**: Protocol for web communication. HTTPS = secure (encrypted).
**NL**: Protocol voor webcommunicatie. HTTPS = veilig (versleuteld).

---

# I

### Index (Database)
**EN**: Database optimization that speeds up searches.
**NL**: Database optimalisatie die zoekopdrachten versnelt.

### InnoDB
**EN**: MySQL database engine that supports foreign keys.
**NL**: MySQL database engine die foreign keys ondersteunt.

---

# J

### JavaScript (JS)
**EN**: Programming language that runs in browsers.
**NL**: Programmeertaal die draait in browsers.

### JSON
**EN**: JavaScript Object Notation - data format for exchanging data.
**NL**: Data formaat voor het uitwisselen van gegevens.

### Junction Table
**EN**: Table that connects two other tables in a many-to-many relationship.
**NL**: Tabel die twee andere tabellen verbindt in een veel-op-veel relatie.

---

# M

### Many-to-Many (N:M)
**EN**: Relationship where many records in Table A relate to many in Table B.
**NL**: Relatie waarbij veel records in Tabel A relateren aan veel in Tabel B.

### MySQL
**EN**: Popular open-source database management system.
**NL**: Populair open-source database management systeem.

---

# O

### One-to-Many (1:N)
**EN**: Relationship where one record relates to many others.
**NL**: Relatie waarbij één record relateert aan vele anderen.

---

# P

### Parameter
**EN**: Value passed into a function.
**NL**: Waarde doorgegeven aan een functie.

### PDO
**EN**: PHP Data Objects - secure way to connect PHP to databases.
**NL**: Veilige manier om PHP te verbinden met databases.

### PHP
**EN**: Server-side programming language for web development.
**NL**: Server-side programmeertaal voor webontwikkeling.

### Prepared Statement
**EN**: SQL query where user input is separated from the query (prevents SQL injection).
**NL**: SQL query waarbij gebruikersinvoer gescheiden is van de query (voorkomt SQL injectie).

### Primary Key (PK)
**EN**: Unique identifier for each record in a table.
**NL**: Unieke identificatie voor elk record in een tabel.

---

# R

### Regex (Regular Expression)
**EN**: Pattern for matching text. Example: `/^\s*$/` matches empty or spaces-only.
**NL**: Patroon voor het matchen van tekst.

### Return Value
**EN**: Value that a function sends back after running.
**NL**: Waarde die een functie terugstuurt na uitvoering.

---

# S

### Server-side
**EN**: Code that runs on the web server (PHP).
**NL**: Code die draait op de webserver (PHP).

### Session
**EN**: Temporary storage that keeps user logged in between pages.
**NL**: Tijdelijke opslag die gebruiker ingelogd houdt tussen pagina's.

### Soft Delete
**EN**: Marking a record as deleted without actually removing it from database.
**NL**: Een record markeren als verwijderd zonder het daadwerkelijk uit de database te verwijderen.

### SQL
**EN**: Structured Query Language - language for managing databases.
**NL**: Taal voor het beheren van databases.

### SQL Injection
**EN**: Attack where malicious SQL is inserted into queries. Prevented with prepared statements.
**NL**: Aanval waarbij kwaadaardige SQL wordt ingevoegd in queries.

---

# T

### Trim
**EN**: Remove whitespace from start and end of text.
**NL**: Witruimte verwijderen van begin en einde van tekst.

---

# U

### UTF-8 / UTF8MB4
**EN**: Character encoding that supports all languages and emojis.
**NL**: Tekencodering die alle talen en emoji's ondersteunt.

---

# V

### Validation
**EN**: Checking if data is correct before saving.
**NL**: Controleren of gegevens correct zijn voordat ze worden opgeslagen.

### VARCHAR
**EN**: Database data type for variable-length text.
**NL**: Database datatype voor tekst met variabele lengte.

---

# X

### XAMPP
**EN**: Software package containing Apache, MySQL, and PHP for local development.
**NL**: Softwarepakket met Apache, MySQL, en PHP voor lokale ontwikkeling.

### XSS (Cross-Site Scripting)
**EN**: Attack where malicious JavaScript is injected into a website. Prevented with htmlspecialchars().
**NL**: Aanval waarbij kwaadaardige JavaScript wordt geïnjecteerd in een website.
