## üìå Projectoverzicht

De **GamePlan Scheduler** is een krachtige, veilige en responsieve web-applicatie die gamers helpt om hun sociale en technische gaming-leven centraal te beheren. In een wereld waar speelsessies vaak ad-hoc via chat worden geregeld, biedt GamePlan de nodige structuur en visuele overzicht.

### üç± Technologische Stack
- **Backend**: PHP 8.1 (Gecentraliseerde logica in `functions.php`)
- **Database**: MariaDB 10.4 (Relationeel 3NF schema met InnoDB)
- **Frontend**: HTML5, CSS3 (Glassmorphism), Vanilla JavaScript
- **Security**: PDO Prepared Statements, Bcrypt Hashing, HART-Protocol

## üéñÔ∏è EXAM DEFENSE CHEAT SHEET (Snelvuur-Antwoorden)

- **SQL-Injectie?** Voorkomen via PDO Prepared Statements (geen strings meer direct in queries).
- **XSS?** Voorkomen via `safeEcho()` die `htmlspecialchars` toepast op elke output.
- **Wachtwoorden?** Nooit in plain-text; altijd gehasht met `password_hash()` (Bcrypt).
- **Sessies?** Beveiligd met `HttpOnly` en `SameSite: Strict` vlaggen tegen kaping.
- **Normalisatie?** De database is in 3NF (Derde Normaalvorm) om data-redundantie te elimineren.
